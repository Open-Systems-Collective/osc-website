# Projects

## Open Knowledge Blocks (OKBs)

**Open Knowledge Blocks (OKBs)** is the flagship project of the Open Systems Collective. It is a programmable governance infrastructure that compiles regulatory obligations into machine-checkable SHACL constraints over RDF evidence graphs.

OKBs enable organizations and developers to encode compliance requirements as structured, executable knowledge blocks. These blocks can be validated automatically against semantic evidence, bridging the gap between regulatory intent and technical enforcement.

### Key Capabilities

- **Regulatory Compilation:** Translate regulatory obligations into formal, machine-checkable constraints.
- **SHACL Validation:** Leverage SHACL (Shapes Constraint Language) to define and enforce data shape constraints over RDF graphs.
- **Evidence Graphs:** Validate compliance claims against structured RDF evidence graphs.
- **Profile-Based Compliance:** Support modular, reusable compliance profiles that can be composed and extended.

### EK-BLOCKS Prototype

The reference implementation is available as the **EK-BLOCKS** prototype, a semantic and logic-based framework for responsible AI reasoning.

- **Repository:** [github.com/Open-Systems-Collective/ek-blocks](https://github.com/Open-Systems-Collective/ek-blocks)

EK-BLOCKS provides a working demonstration of how ontological knowledge blocks can be used to achieve executable, profile-based compliance for trustworthy AI services.
